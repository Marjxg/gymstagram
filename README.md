Gymstagram es un proyecto hecho con Laravel y TailwindCSS.

## Ambiente de desarrollo
* Docker desktop
* WSL
* Ubuntu

## Laravel
Laravel es un framework de aplicaciones web basado en objetos

### Comandos útiles
```bash
sail up
sail down
```
## Tailwind CSS
Estilos CSS y apariencia para el proyecto.

Instalación:
```bash
sail npm install -D tailwindcss postcss autoprefixer
sail npx tailwindcss@3 init -p
sail npm run dev
```
